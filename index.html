‚ùìFrequently Asked Questions

Battery Sentinel not visible in menu bar

Troubleshooting steps:

  1. Verify Battery Sentinel is running in Activity Monitor
  2. Quit and restart the application completely
  3. Check for hidden menu bar icons (expand menu bar)
  4. Restart your Mac if issue persists

Customizing warning thresholds

Configuration process:

  1. Click Battery Sentinel icon in menu bar
  2. Select Preferences... from dropdown
  3. Adjust threshold sliders for your workflow
  4. Click Save Settings to confirm

Launch at login not working

Setup instructions:

  1. Access Battery Sentinel menu in menu bar
  2. Ensure Launch at Login is enabled
  3. Manual setup: System Settings ‚Üí General ‚Üí Login Items
  4. Add Battery Sentinel to startup applications

Testing warning functionality

Preview your settings:

  1. Access Battery Sentinel menu bar menu
  2. Select Test Low Warning or Test Critical Warning
  3. Review preview alert appearance and timing
  4. Adjust preferences based on preview results

Disabling specific warning types

Selective notification control:

  • Open Preferences from menu bar menu
  • Toggle off specific warning types as needed
  • Low and critical warnings can be controlled independently
  • Settings are saved automatically

No warnings while charging

This is intelligent design. Battery Sentinel automatically detects charging status and pauses warnings when your Mac is plugged in. Alerts resume when unplugged.

To completely uninstall Battery Sentinel:

  2. Move application to Trash from Applications folder
  3. Remove from Login Items: System Settings ‚Üí General ‚Üí Login Items
  4. Restart Mac to complete removal process
